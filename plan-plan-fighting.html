优势 & 预警

✦ 核心优势

  • 长期目标拆解为阶段计划
  • 周计划与每日行动实时联动
  • 任务、财务、生活和自定义模块统一管理
  • 本地优先保存，支持导入导出和同步
  • 可按年、月或自定义栏目扩展

△ 风险预警

  • 目标过多导致执行分散——每阶段保留核心重点
  • 长期计划容易脱离现实——每月复盘并调整
  • 时间、财务和生活目标可能互相挤压
  • 关键路径要保留备选方案

📖 使用说明书

🎯 总览

这是一个本地优先的多页面计划系统。首页用于通用全景规划，顶部页签可新增按年或按月组织的大计划；左侧工具栏提供编辑、备份、同步、月历、财务、学术、购房、求职、城市和记录本等模块。

  • 数据默认保存在浏览器本地存储，刷新后会自动恢复
  • 周计划、月历提醒、工具面板数据和地图备注均走本地保存
  • 页面主题、字号、字体和布局会跟随设置实时调整

✎ 全局编辑模式

点击左侧 「编辑」 或按 E 进入全局编辑模式。进入后，页面内标题、副标题、分段标题、卡片图标、卡片标题、说明文字、统计栏、页脚和周计划内容都可以直接修改。

  • 动态页面的分段标题会同步到顶部导航，导航和小标题保持一致
  • 英雄区统计栏可编辑；任务、已完成、完成率、月份或年份为基础栏，其他栏目可在编辑模式下新增、删除和修改
  • 失焦后自动保存；左侧各工具面板内也有独立保存按钮
  • 不想继续编辑时再次点击 「编辑」 退出，页面会回到浏览状态

📄 页面与大计划

  • 顶部 ＋ 可新增计划页，并选择按年或按月组织
  • 每个新页面都采用和首页一致的布局：英雄区、分段导航、卡片网格、周计划和实时进度
  • 每个页面可自定义分类，例如学术、锻炼、财务、生活、求职或其他目标
  • 左侧工具栏展开后会占用独立空间，主内容区会让出位置，避免遮挡标题和正文

📅 周计划

  • 每张卡片下方都有周计划，周一至周日每天一条任务
  • 每天可选择时间段：上午、下午、晚上或全天
  • 每日标签显示序号和对应颜色，便于快速选择，不遮挡任务文字
  • 支持勾选完成、拖拽调整顺序、Markdown 预览、每日随笔、周总结和模板复用
  • 点击周标题展开或收起，点击周标题旁 ✕ 删除该周
  • 展开 和 收起 可批量展开或收起所有周计划

🗓️ 月历与今日概览

  • 月历 可按日期添加事项，编辑过的日期会高亮
  • 未来一周内的未完成事项会出现在每次打开页面时的今日概览中
  • 事项适合记录截止日期、会议、考试、看房、旅行和提醒任务

🛠 左侧工具栏

  • ◀ / ▶ 展开或收起工具栏；展开时主内容区会自动避让
  • 章节信息 显示当前阅读位置，点击可跳转当前章节
  • 编辑 切换全局编辑模式
  • 展开 / 收起 批量控制周计划显示状态
  • 存档 将当前数据保存到仓库入口
  • 同步 打开 GitHub Gist 同步配置和状态
  • 安装 在浏览器支持时将网页安装到桌面
  • 说明 打开本说明书
  • 设置 调整字号、字体、行距和宽度；导出、导入、主题和重置也在设置面板内完成
  • 备份 打开备份与恢复面板

💰 财务与快照

  • 财务 包含信用卡还款计算、存款目标和收支记录，编辑后点击面板内保存按钮
  • 快照 可记录当前进度状态，用于比较不同时间点的任务完成变化
  • 财务、快照等工具面板的数据会写入本地数据，刷新后继续保留

🎓 学术、购房、求职与城市

  • 学术 管理论文里程碑、实验日志和文献记录
  • 购房 管理房贷计算、购房成本、看房清单和多方案对比
  • 求职 记录选调、求职和其他职业节点，可按类型筛选
  • 城市 建立城市数据卡，用于比较生活成本、岗位环境、交通、教育、医疗等信息

📓 记录本

记录本 用于保存日常笔记、复盘和阶段性想法。支持多页切换、新增页面、删除页面和手动保存，适合记录周总结、月总结、科研想法、买房观察和求职复盘。

🌍 世界地图

如当前版本启用了地图入口，地图为三维交互式世界地图；中国地图内容已删除，只保留世界地图。

  • 拖拽旋转，滚轮缩放，按钮可放大、缩小和重置
  • 鼠标悬停国家时显示当地时间和基本信息
  • 点击国家后可在右侧编辑备注：时间、事件、景点、计划等
  • 每个国家可新增多页备注，并可翻页查看、保存或删除当前页

🍅 番茄钟与专注

  • 右下角番茄钟支持 15/25/30/45/60/75/90 分钟专注时长
  • 支持短休、长休、浮窗计时、拖拽位置、专注标签、统计图表和里程碑
  • 睡眠模式可进入全屏专注界面，减少干扰

🤖 AI 助手

  • 右下角 AI 按钮可拖拽移动，点击打开对话面板
  • 可用自然语言提出任务修改、添加周计划、调整目标和总结计划
  • AI 修改会先生成建议，确认后才写入数据
  • 首次使用需在 AI 面板中配置兼容接口的 API Key

💾 数据与同步

  • 普通编辑会自动保存到浏览器本地存储
  • 工具面板中的保存按钮用于立即收集当前面板输入并写入数据
  • 导出 生成 JSON 备份；导入 从备份恢复
  • 同步 使用 GitHub Gist 跨设备同步，需要先保存 Token
  • 存档 用于仓库数据归档；代码版本仍通过 Git 管理

🖨 打印与性能

  • 打印会隐藏工具栏、弹窗、背景粒子和悬浮控件，保留正文内容
  • 页面已优化滚动、统计刷新和背景动画；切到后台时会暂停粒子和三维背景
  • 如果页面变慢，可先收起大量周计划，或关闭不需要的弹窗和面板

⌨️ 快捷键

  • Ctrl+F 全文搜索
  • E 切换编辑模式
  • Ctrl+S 手动保存
  • ? 打开或关闭说明书
  • Space 勾选下一项待办
  • F 专注模式切换
  • Esc 关闭当前弹窗
