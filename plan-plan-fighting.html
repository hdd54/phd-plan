📖 使用说明书

🎯 这是什么？

博士五年规划 + 买房作战计划表。按年规划学业、备考、财务和生活目标，支持周级任务管理、进度追踪、AI 辅助编辑。

✎ 编辑模式

点击底部工具栏 「编辑」 按钮切换编辑模式：

  • 编辑模式下可以修改卡片标题、描述文字、列表内容
  • 周计划中可编辑每天的任务、勾选框标记完成
  • 点击 「展开」 可展开所有周计划
  • 点击周标题旁的 「✕」 可删除该周
  • 任意周下方 「＋ 添加周」 可新增一周

📊 年度结构

页面按五年规划分为五个区域，每一年包含四个卡片：

  • 🎓 学业 — 论文、科研、毕业进度
  • 📝 备考 — 考公/选调准备
  • 💰 财务 — 存款、收入、买房预算
  • 🏠 生活 — 健康、日常安排

每张卡片底部有进度条，显示该目标完成比例。

📅 周计划

每个年度下方有周计划区域（周一至周日）。点击周标题可展开/收起该周详情，每天可填写任务内容并勾选完成。周标题右侧显示该周完成进度。

🤖 AI 助手

右下角圆形按钮打开 AI 面板。支持：

  • 自然语言修改任务内容
  • 添加/删除周计划
  • 调整年度目标
  • 多轮对话管理（📁 按钮查看历史对话）

首次使用需在 AI 面板中填入 API Key（支持 DeepSeek 等兼容接口）。

⌨️ 快捷键

  • Ctrl+F — 全文搜索
  • E — 切换编辑模式
  • Ctrl+S — 手动保存
  • ? — 打开本说明书

💾 数据管理

所有数据自动保存在浏览器本地存储中。底部工具栏支持：

  • 📥 导出 — 下载 JSON 备份文件
  • 📤 导入 — 从备份文件恢复数据
  • ⚡ 同步 — 通过 GitHub Gist 跨设备同步（需配置 Token）
  • 🗑 重置 — 清空所有数据恢复默认
