📖 使用说明书

🎯 这是什么？

博士规划 + 多页面作战计划表。按年或按月规划学业、备考、财务、生活、购房、求职等目标，支持周级任务、实时进度、标签、月历提醒、AI 辅助编辑和番茄专注计时。

✎ 编辑模式

点击左侧工具栏 「编辑」 按钮切换编辑模式：

  • 编辑模式下可以修改首页和自定义页面的标题、副标题、分段标题、卡片标题、描述文字、列表内容和页脚
  • 周计划中可编辑每天的任务、勾选框标记完成
  • 每日任务左侧可选择标签颜色；周计划显示标签序号和颜色，便于快速识别
  • 点击 「展开」 可展开所有周计划
  • 点击周标题旁的 「✕」 可删除该周
  • 任意周下方 「＋ 添加周」 可新增一周，也可使用 「📋 模板」 批量创建

📊 年度结构

默认首页按博士五年规划分为五个区域，每一年包含四个卡片：

  • 🎓 学业 — 论文、科研、毕业进度
  • 📋 备考 — 考公/选调准备
  • 💰 财务 — 存款、收入、买房预算
  • ❤️ 生活 — 健康、日常安排

每张卡片底部有进度条，显示该目标完成比例。顶部统计会随任务总数和完成数实时变化。

📄 多页面计划

顶部页签可以新增计划页。新页面可选择按年或按月组织，并自定义分类。每个页面会使用与博士计划一致的英雄区、分段导航、卡片网格、周计划和进度条布局；顶部任务数、完成数、完成率和年份/月数会随编辑、勾选、增删周计划实时刷新。

📅 周计划

每个年度或月份下方有周计划区域（周一至周日）。点击周标题可展开/收起该周详情，每天可填写任务内容并勾选完成。周标题右侧显示该周完成进度。支持拖拽排序、Markdown 预览、每日随笔、周总结和模板复用。

🗓️ 月历与提醒

月历可按日期添加事项。已添加事项的日期会高亮，未来一周内未完成事项会在每次打开页面时显示到今日概览中，便于提前处理。

🍅 番茄钟

点击右下角番茄钟按钮打开番茄钟面板：

  • 支持 15/25/30/45/60/75/90 分钟专注时长
  • 5 分钟短休和 15 分钟长休
  • 浮窗计时器可自由拖拽到任意位置
  • 自动记录专注历史，提供统计图表和里程碑
  • 睡眠模式：点击 🌙 进入全屏专注
  • 每个番茄可添加标签（如：论文、刷题）

📓 记录本

左侧工具栏 「记录本」 打开周期回顾面板，支持周/月/年维度的学习复盘、进度记录和心得笔记。

🎓 学术与职业

学术 面板整合论文里程碑、实验日志和文献管理；求职 管理选调/求职时间线；城市 用于记录和对比目标城市信息。

🏠 购房与财务

购房 支持贷款、成本和看房清单；财务 支持信用卡还款、存款目标和收支记录；房贷多方案对比可用于比较不同首付、利率和年限。

🤖 AI 助手

右下角圆形按钮打开 AI 面板，支持拖拽移动位置。支持：

  • 自然语言修改任务内容
  • 添加/删除周计划
  • 调整年度目标
  • 多轮对话管理（📁 按钮查看历史对话）

首次使用需在 AI 面板中填入 API Key（支持 DeepSeek 等兼容接口）。

⌨️ 快捷键

  • Ctrl+F — 全文搜索
  • E — 切换编辑模式
  • Ctrl+S — 手动保存
  • ? — 打开本说明书
  • Space — 勾选下一项待办
  • F — 专注模式切换
  • Esc — 关闭当前弹窗

🛠 左侧工具栏

  • ✎ 编辑 — 修改页面文字、卡片内容和计划说明
  • 📂 展开 / 📁 收起 — 批量展开或收起所有周计划
  • 📥 导出 / 📤 导入 — 导出或导入 JSON 备份
  • 💾 存档 — 保存当前数据到仓库入口
  • ⚡ 同步 — 查看数据同步状态并打开同步设置
  • 💰 财务 — 信用卡还款计算 + 存款目标管理
  • 📸 快照 — 拍摄进度快照，对比变化
  • 🌙 主题 — 切换明/暗主题
  • 💾 备份 — 数据备份与恢复
  • 🗓️ 月历 — 月历视图和一周内事项提醒
  • 🎓 学术 — 论文/实验/文献管理
  • 🏠 购房 — 贷款计算 + 购房成本清单
  • 📋 求职 — 选调/求职时间线
  • 🏙️ 城市 — 城市数据卡对比
  • 📓 记录本 — 周期回顾与学习笔记
  • 📖 说明 / ⚙ 设置 — 打开说明书或调整显示设置
  • 🗑 重置 — 清空任务数据恢复默认

💾 数据管理

所有数据自动保存在浏览器本地存储中。左侧工具栏支持：

  • 📥 导出 — 下载 JSON 备份文件
  • 📤 导入 — 从备份文件恢复数据
  • ⚡ 同步 — 通过 GitHub Gist 跨设备同步（需配置 Token）
  • 🗑 重置 — 清空所有数据恢复默认
  • 💾 存档 — 保存数据到仓库
  • GitHub 推送 — 代码改动仍通过本仓库版本管理
